# Tasteatlas-simple-page-scraper
Scrapes the name of recipes and the rating from tasteatlas pages, then downloads them as a json file. 

Warning: this is only for scraping one page at a time, manually, if you just want to grab a list of recipes from a top 100 page or something, for example. 

Simply copy the code from the javascript file and open the tasteatlas page with recipes, then right click and "inspect element" or the equivalent for your browser. From there, paste the code in the console tab that opens at the bottom.

The CSV exporter creates a master file in a cleaner format. 

Use whichever is best for you. Or don't, because this only scrapes one page at a time. 

That's all I needed, but if you want to add functionality for multiple pages, feel free to fork.
